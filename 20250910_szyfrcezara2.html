Szyfr Cezara (OOP w JS + Python)

Podaj tekst, wybierz przesunięcie i zaszyfruj/odszyfruj. Litery są szyfrowane, spacje i znaki specjalne zostają. Poniżej znajdziesz też przykładową implementację w Pythonie.

Uwaga: przesunięcie większe niż 25 zostanie zredukowane modulo 26.

Przykładowy kod w Pythonie


def szyfruj_cezara(tekst, klucz): zaszyfrowany_tekst = "" for litera in tekst: if 'a' <= litera <= 'z': start = ord('a') nowa_pozycja = (ord(litera) - start + klucz) % 26 zaszyfrowany_tekst += chr(start + nowa_pozycja) elif 'A' <= litera <= 'Z': start = ord('A') nowa_pozycja = (ord(litera) - start + klucz) % 26 zaszyfrowany_tekst += chr(start + nowa_pozycja) else: zaszyfrowany_tekst += litera return zaszyfrowany_tekst

tekst_jawny = input("Podaj tekst do zaszyfrowania: ") przesuniecie = int(input("Podaj klucz przesunięcia (np. 3): ")) zaszyfrowany = szyfruj_cezara(tekst_jawny, przesuniecie) print("Zaszyfrowany tekst:", zaszyfrowany)

Deszyfrowanie:

odszyfrowany = szyfruj_cezara(zaszyfrowany, -przesuniecie)
