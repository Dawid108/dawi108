Analiza Tablic: PHP vs JavaScript

1. Tablice Indeksowane Numerycznie

W PHP tablice tworzy się przez array() lub []. JavaScript używa wyłącznie []. Obie technologie pozwalają na przechowywanie różnych typów danych w jednej tablicy.
// PHP: $kolory = array("czerwony", "zielony");
// JS: const kolory = ["czerwony", "zielony"];

2. Tablice Asocjacyjne (Klucz => Wartość)

W PHP tablica asocjacyjna to wciąż array. W JavaScript "tablice z kluczami tekstowymi" realizuje się za pomocą Obiektów {} lub Map. Przebiegamy po nich pętlą for...in lub Object.entries().

3. Tablice Dwuwymiarowe (Macierze)

Zarówno w PHP, jak i JS, macierz to tablica zawierająca inne tablice. Do sumowania elementów używamy zagnieżdżonych pętli.

4. Filtrowanie i Agregacja Danych

Przykład z samochodami pokazuje, jak wyciągać konkretne dane (np. spalanie) z obiektów i wykonywać na nich operacje logiczne (if) oraz matematyczne (suma mocy).
