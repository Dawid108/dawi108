Analiza Skryptu: Symulacja Kieszonkowego

1. Oryginalny kod PHP

Kod pobiera dane z adresu URL (metoda GET) i oblicza, po ilu miesiącach suma odłożonych pieniędzy przekroczy założony limit.

$kwota_pocz = $_GET['kwota_pocz']; // startowa kwota co miesiąc
$wzrost = $_GET['wzrost'];         // o ile rośnie co miesiąc
$kwota_konc = $_GET['kwota_konc']; // cel (suma oszczędności)
$kwota = $kwota_pocz;              // aktualna stawka miesięczna
$suma = $kwota_pocz;               // całkowite oszczędności
$nr = 1;                           // licznik miesięcy

while($suma < $kwota_konc){
    echo "$nr $kwota $suma<br>";
    $kwota = $kwota + $wzrost;     // zwiększ stawkę
    $suma = $suma + $kwota;        // dodaj do skarbonki
    $nr++;
}
echo "$nr $kwota $suma<br>";        // ostatni krok po przekroczeniu celu

2. Jak to działa? (Wyjaśnienie)

  • Pętla while: Działa tak długo, aż uzbierana $suma jest mniejsza niż $kwota_konc.
  • Logika wzrostu: To nie jest tylko odkładanie tej samej kwoty. Co miesiąc samo "kieszonkowe" ($kwota) rośnie o wartość $wzrost.
  • Wyświetlanie: Skrypt wypisuje numer miesiąca, aktualną stawkę miesięczną oraz stan konta po dodaniu wpłaty.
