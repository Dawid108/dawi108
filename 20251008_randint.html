Zgadnij liczbę (0–10)

Program losuje liczbę, Ty zgadujesz. Poniżej jest też gotowy kod Pythona do skopiowania.

Losuje jedną liczbę całkowitą od 0 do 10. Masz nieograniczoną liczbę prób.
Twoja kolej — spróbuj zgadnąć!
Wciśnij "Zgaduję" aby sprawdzić.
Powodzenia!
Próby: 0
Zakres: 0–10
Terminal: kod Pythona 
import random  # import biblioteki random

losujemy liczbę całkowitą z zakresu 0–10

wylosowana = random.randint(0, 10)

próby = 0

while True: próby += 1 try: wej = int(input("Zgadnij liczbę od 0 do 10: ")) except ValueError: print("Proszę podać liczbę całkowitą.") continue

if wej < 0 or wej > 10:
    print("Liczba poza zakresem! Podaj 0–10.")
    continue

if wej == wylosowana:
    print(f"Brawo! Zgadłeś liczbę w {próby} próbach!")
    break
elif wej < wylosowana:
    print("Nie zgadłeś — za mało.")
else:
    print("Nie zgadłeś — za dużo.")
