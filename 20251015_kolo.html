Napisz program który oblicza pole i obwód koła o promieniu podanym przez użytkownika, wartość Pi należy pobrać z biblioteki math, wartość promienia nie może być ujemna i trzeba to zabezpieczyć odpowiednim komunikatem, wygeneruj ilustrację prezentującą koło ze skalą x, y.

Rozwiązanie w JavaScript

Kod w Pythonie:

import math
import matplotlib.pyplot as plt

r = float(input("Podaj promień koła: "))

if r < 0:
    print("Błąd: promień nie może być ujemny!")
else:
    pole = math.pi * r ** 2
    obwod = 2 * math.pi * r
    print(f"Pole koła: {pole:.2f}")
    print(f"Obwód koła: {obwod:.2f}")

    fig, ax = plt.subplots()
    circle = plt.Circle((0, 0), r, color='lightblue', fill=True, alpha=0.5)
    ax.add_artist(circle)
    ax.set_xlim(-r * 1.2, r * 1.2)
    ax.set_ylim(-r * 1.2, r * 1.2)
    ax.set_aspect('equal')
    plt.axhline(0, color='black')
    plt.axvline(0, color='black')
    plt.title(f"Koło o promieniu r = {r}")
    plt.show()
