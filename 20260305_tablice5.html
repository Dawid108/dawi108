Analiza i Przepisanie Kodu

// JavaScript (odpowiednik Twojego PHP) const kolory = ["czerwony", "pomarańczowy", "żółty", "zielony"]; console.log("Pierwszy kolor: " + kolory[0]); kolory[4] = "niebieski"; kolory[5] = 22; kolory[6] = 3.141593; kolory[7] = 'w'; // Pętla for for(let i=0; i < kolory.length; i++) { console.log(kolory[i]); } // Tablica asocjacyjna (Obiekt w JS) const RGB = { "R": "czerwony", "G": "zielony", "B": "niebieski" };

Co się dzieje w kodzie?

  • $kolory = array(...): W JS używamy nawiasów kwadratowych []. Tablice są dynamiczne – mogą przechowywać różne typy danych (stringi, liczby) jednocześnie.
  • $kolory[4] = "niebieski": Dodajemy nowy element na konkretnym indeksie. Jeśli indeks 4 nie istniał, tablica się powiększy.
  • count($cyfry): W JS używamy właściwości .length, aby sprawdzić długość tablicy.
  • Tablice Asocjacyjne: PHP pozwala na klucze tekstowe w tablicach. W JavaScript do tego celu służą Obiekty {}.
