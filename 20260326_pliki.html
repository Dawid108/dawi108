Analiza kodu PHP: Odczyt Plików

Poniżej znajduje się analiza trzech metod odczytu pliku 20260326_kotek.txt użytych w Twoim skrypcie.

// Metoda 1: fpassthru
$dp = fopen('20260326_kotek.txt', 'r');
$liczba_bajtow = fpassthru($dp);
fclose($dp);

// Metoda 2: readfile
$lb = readfile('20260326_kotek.txt');

// Metoda 3: file + nl2br
$wiersze = file('20260326_kotek.txt');
foreach ($wiersze as $wiersz){
    $wiersz2 = nl2br($wiersz);
    echo $wiersz2;
}

1. Funkcja fpassthru()

Wymaga wcześniejszego otwarcia pliku przez fopen(). Odczytuje dane od aktualnego miejsca wskaźnika do końca pliku i wypisuje je bezpośrednio do bufora wyjściowego.

Zwraca: liczbę przeczytanych bajtów.

2. Funkcja readfile()

Najprostsza metoda. Nie potrzebujesz fopen(). Otwiera plik, wypisuje zawartość i natychmiast go zamyka.

Zaleta: Najszybsza przy przesyłaniu dużych plików bezpośrednio do przeglądarki.

3. Funkcja file() i nl2br()

file() wczytuje cały plik do tablicy, gdzie każdy element to jedna linia. nl2br() zamienia znaki nowej linii (\n) na znacznik HTML <br>, co pozwala zachować formatowanie tekstu w przeglądarce.
